Задание:

5.  Звукозапись. Определить  иерархию  музыкальных  композиций.  Записать
на диск  сборку.  Подсчитать  продолжительность.  Провести  перестановку
композиций диска на основе принадлежности к стилю. Найти композицию,
соответствующую заданному диапазону длины треков.


Есть 2 интерфейса: Playable, его методами должны обладать все композиции,
и Singable, который реализует только класс VocalComposition, т.к. там есть дополнительный метод sing(), который
должен быть только у композиций с вокалом (и который бессмысленен для инструментальных композиций).
Музыкальный трек может быть или экземляром класса Composition, или, в более узком случае, экземпляром класса
VocalComposition.

У Composition есть несколько полей: имя артиста, название трека, длительность и стиль. Чтобы треку можно было задать
не любой стиль, а только какой-либо из ограниченного набора, я сделал стиль enum'ом.
Методы по работе с композициями я разместил в классе MusicUtils. Они статические, чтобы не было нужды создавать объект
этого класса
MusicMenu содержит main() метод, там же создаётся начальный список композиций.

По заданиям:

1."Записать  на диск  сборку"
Метод recordToDisk() записывает объекты в тектовый файл. Чтобы объекты представлялись в удобочитемом виде, в классе
Composition переопределён метод toString(). Имя тектового файла вводится с клавиатуры

2."Подсчитать  продолжительность"
Метод calculateDuration() подсчитывает общую продолжительность в секундах переданного ему списка треков. Для
представления этого времени в удобочитаемом виде mm:ss используется метод durationFormatter()

3."Провести  перестановку композиций диска на основе принадлежности к стилю"
В классе Composition был реализован интерфейс Comparable и его метод compareTo().Поэтому путём вызова Collections.sort()
можно отсортировать переданную методу arrangeByStyle() коллекцию. Сортировка проводится по стилю, при совпадении
стилей - по имени исполнителя, если совпадает и имя, то по названию трека. После вызова arrangeByStyle() полученный
список композиций выводится в файл с помощью метода recordToDisk()

4."Найти композицию, соответствующую заданному диапазону длины треков"
Метод findByDuration() принимает на вход границы диапазона продолжительности в секундах и список композиций, и
далее печатает в консоль список композиций, удовлетворяющих по продолжительности введённым пользователем значениям
